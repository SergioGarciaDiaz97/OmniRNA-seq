# OmniRNA-seq
OmniRNA-seq es un ecosistema bioinformático integral para el análisis automatizado y reproducible de datos de RNA seq bulk en entornos HPC. Transforma lecturas crudas de secuenciación en resultados biológicos, desacoplando la ingeniería de datos del modelado estadístico y del despliegue reproducible basado en contenedores Apptainer/Singularity.
